# Secure Software Development Project

## Description
This project is a secure web application developed using Django.
It implements OWASP-compliant secure coding practices.

## Features
- User registration and login
- Role-based access control (Admin / User)
- Secure CRUD module
- Audit logging

## How to Run
1. Clone the repository
2. Create virtual environment
3. Install dependencies
4. Run the server

## Security Controls
- CSRF protection
- Password hashing
- Input validation
- Access control enforcement

## Technologies
- Python
- Django
- SQLite
